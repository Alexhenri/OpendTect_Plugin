6.1.9 Volume Builder Output

Volume-builder creator window can be launched from Processing > Create Seismic Output > Volume Builder. It is used to create the output volume that has been defined in a volume builder setup. Optionally, if the initial volume builder setup is not defined, press the Edit button to define the setup. Press 'Ok' to launch a batch processing window.
