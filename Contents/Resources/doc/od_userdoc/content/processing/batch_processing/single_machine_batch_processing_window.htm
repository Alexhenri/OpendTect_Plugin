6.6.1 Single Machine Batch Processing Window

In single mode processing, the data can be processed either on a local machine or on a remote host. All relevant information on the progress of the calculation will be stored, (see Job information file).

'Execute remote' toggled on to send the job to a remote machine. The job priority can be changed (-19-lowest to +19 highest).
