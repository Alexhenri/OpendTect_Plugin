4.5.2.1 Body Operator

The body operator, accessed from the icon in the Bodies manager, enables to perform various operations on a geological body (or combinations thereof), as listed below.

For all the operators, click on the Input to select either a saved body or create a body using an operator. Each operation needs 2 inputs. You can combine more than one input using an operator as input. After giving a name to the new body, it will be created after clicking on Run.
