4.5.3 Manage Color Tables

The Color Table Manager gives access to the various settings used for data visualization to edit an existing colour table or create a new one.

The manager is accessible from Survey > Manage > Color Tables and from the right-click menu on the colour table in the 3D scene (See below).

For a full description of the various options and possibilities, please see the following section: Color Tables
