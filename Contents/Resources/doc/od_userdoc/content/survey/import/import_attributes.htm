4.3.1 Import Attributes

An OpendTect attribute set can be imported via Survey > Import > Attribute > ASCII....

An OpendTect attribute set file contains a set of attribute definitions created in the Attribute Set window. OpendTect attribute sets are stored in ../'Survey Data Root folder'/Attribs/ and have '.attr' extensions.

In the Import Attribute Set window: locate an OpendTect attribute set, and provide an Output Attribute Set name to be used in the current project.

The imported attribute set can then be opened within the Attribute Set window.

More options for importing attribute sets are available in the Attribute Set window itself.
