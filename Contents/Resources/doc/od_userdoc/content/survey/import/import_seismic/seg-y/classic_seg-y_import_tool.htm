4.3.10.1.2 Classic SEG-Y Import Tool

The Classic SEG-Y Import Tool used to be the main utility for import of SEG-Y data prior to OpendTect 6.0. This wizard can still be launched via:

  • Survey > Import > Seismics > SEG-Y > Classic tool;
  • from the SEG-Y Wizard introduced in OpendTect 6.0 by clicking on 'Classic'.

It allows to import 2D and 3D pre- and poststack seismic data from one or multiple SEG-Y files. The import consists of two mandatory steps: SEG-Y Import Preparation and SEG-Y Import itself.
