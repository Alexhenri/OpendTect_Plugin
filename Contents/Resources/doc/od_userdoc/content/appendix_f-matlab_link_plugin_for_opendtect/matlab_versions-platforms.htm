16.2 MATLAB Versions/Platforms

The plugin is available in the Linux64 and Windows64 versions of OpendTect. MATLAB R2013a has been used to build the libraries and is therefore the recommended version to build the shared libraries. Versions of MATLAB older than R2013a may not work with this plugin.
