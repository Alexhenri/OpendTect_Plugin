7.3 Wheeler Scenes

The Wheeler Scene is a transformation (flattening) of HorizonCube into relative geological time (RGT). Therefore, before adding a Wheeler Scene, the HorizonCube will need to be selected. You will be prompted for this if not already selected.

This option is only available if the SSIS plugin is installed.
