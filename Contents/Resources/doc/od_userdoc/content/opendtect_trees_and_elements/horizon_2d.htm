3.8 2D Horizon

Add... an existing 2D horizon into the scene.

Track new... Allows a new 2D horizon to be created.

Create from 3D... Generate 2D horizons from existing 3D horizons by right-clicking on 2D Horizon in the tree and selecting Create from 3D. A window pops up where you can designate the 3D horizon, the 2D data set and 2D line(s) where you want to create the new 2D horizon. This function allows for the 2D horizon to be created on one line, or on as many lines as a data set contains, all at once.
