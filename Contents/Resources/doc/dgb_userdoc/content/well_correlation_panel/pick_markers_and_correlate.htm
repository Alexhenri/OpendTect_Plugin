6.4 Pick Markers and Correlate

New well-markers are defined or alternatively, the existing markers can be modified using WCP. The purpose of this functionality is to build a new stratigraphic framework of the area. The Edit Makers dialogue is launched by clicking on this icon.

How to pick markers?

By keeping this dialog box open, markers are added to the well using a mouse click on the well-panel displayed in the WCP. Before starting picking, the 'interact' (Pick) mode should be toggled on . Once markers are interpreted and added to all wells, press the OK/Save button to save the markers.

This dialog box is used when picking new markers.
