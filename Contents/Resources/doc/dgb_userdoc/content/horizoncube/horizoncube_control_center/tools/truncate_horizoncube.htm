4.2.3.13 Truncate HorizonCube

An existing continuous HorizonCube often contains very closely spaced events. The HorizonCube can later be filtered by defining a maximum density (threshold) as a number of closely spaced events. In the following window the data selection is very simple. Select the input HorizonCube (continuous) in the input field and type the output HorizonCube name in the output field. The Maximum density defines a threshold for two horizons in the HorizonCube. The value '1' defines a threshold of one event per sample of a seismic data. Optionally, the area sub-selection can also be made to do the conversion within a sub-volume only.

To learn more about the benefit of applying a density filter to the continous HorizonCube, please read the Wheeler Transformation.
