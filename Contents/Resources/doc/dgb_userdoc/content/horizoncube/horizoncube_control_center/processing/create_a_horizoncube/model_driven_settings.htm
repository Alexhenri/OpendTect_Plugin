4.2.1.3.1 Model Driven Settings

In the Specify Parameters window a user can choose between two sampling methods for the HorizonCube events:

  • Spacing at maximum thickness, for which the spacing parameter (in ms / m /ft) needs to be specified. This sets the sampling rate of the data in the Wheeler domain.
  • Number of events that defines the number of Chronostratic events that can be exported as horizons.

The option "Apply to all sequences" takes effect only when pressing "OK" and cannot be undone.
