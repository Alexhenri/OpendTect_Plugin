7.4.8 Dump Model to File

All the pseudo-well models forming a profile simulation can be saved to a .txt file using this option available in the right-click menu.

If there are some layer quantities present in the pseudo-well models that are computed from other quantities using Math formulas, it is possible to save those formulas as well in the output dump file.
