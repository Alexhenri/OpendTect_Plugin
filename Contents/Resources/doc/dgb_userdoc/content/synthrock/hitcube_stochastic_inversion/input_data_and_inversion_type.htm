7.6.1 Input Data and Inversion Type

The input window firstly allows you to chose the inversion type. The first one is the mode which proceeds the scoring regarding a selected horizon. The second one proceeds the scoring within the entire volume.

Then, before selecting any input data, you have to select the type of this one if it is a Lineset (2D) or a Cube (3D). And finally, the synthetic which will be used to match the data has to be selected, if your choice is not the default one.

When these fields are selected click on Ok to go to the Analysis window.
