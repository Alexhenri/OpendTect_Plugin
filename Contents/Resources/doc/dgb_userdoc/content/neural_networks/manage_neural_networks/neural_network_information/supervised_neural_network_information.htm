8.2.3.1 Supervised Neural Network Information

The information window shows neural network information for the training set and the test set. For each set the numbers of usable and invalid vectors are shown.

Balancing is automatically performed between the data extraction and the training. This ensures that there is an equal representation of all inputs picksets. Misclassification information is displayed based on the membership to the input pickset.
