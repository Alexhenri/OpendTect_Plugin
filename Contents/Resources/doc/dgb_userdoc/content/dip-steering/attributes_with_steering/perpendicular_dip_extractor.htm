3.5.8 Perpendicular Dip Extractor

This attributes works similarly to the volume statistics attribute, with one very important exception:

The time gate used for the extraction of the amplitudes is not vertical, but is perpendicular to the layer dip. This layer dip is read from a SteeringCube, which is why it is not possible to toggle off the full steering for this attribute.

Using this attribute instead of the standard volume statistics will most often return more "correct" amplitudes, because the extraction is done in the same geological layer with a real constant thickness, whereas a vertical time gate would represent an apparent thickness, varying through the survey.
