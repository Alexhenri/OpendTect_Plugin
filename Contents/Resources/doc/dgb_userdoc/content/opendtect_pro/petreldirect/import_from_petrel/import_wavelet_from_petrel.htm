1.1.2.5 Import Wavelet From Petrel

Petrel wavelets can be accessed in OpendTect by clicking on PetrelDirect insert icon in the following places:

  • Manage Wavelets window (Survey > Manage > Wavelets or click on icon in the Manage Data toolbar)

Select Wavelet(s) window allows to select wavelets from Petrel project and create either Link to Petrel or OpendTect Copy.
