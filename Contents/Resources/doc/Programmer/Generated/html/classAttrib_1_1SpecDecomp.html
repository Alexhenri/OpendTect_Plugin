Detailed Description

Spectral Decomposition Attribute.

Calculates the frequency spectrum of a trace

SpecDecomp gate=[-12,12] window=[Box]|Hamming|Hanning|Barlett|Blackman|CosTaper5
Input:
0       Real data
1       Imag data
Output:
0       Spectrum 0 freq
1       freq 1
2       freq 2
|       etc
|       etc
