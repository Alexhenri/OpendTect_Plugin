float Well::getDefaultVelocity ( )

If survey unit is depth-feet, it returns the equivalent of 8000 (ft/s), otherwise 2000 (m/s). Its purpose is to get nice values of velocity where we may need such functionality ( eg. replacement velocity for wells ).
