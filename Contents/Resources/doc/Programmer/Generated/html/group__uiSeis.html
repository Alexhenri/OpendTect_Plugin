Detailed Description

Seismics User Interface

Most of the tools in this module enable import and export of seismic data into/from the OpendTect data store. The merge and manage tools manipulate seismic data already in the data store.

A standalone 'application' is the uiSeisMMProc class, which handles the distribution of processing on multiple machines.
