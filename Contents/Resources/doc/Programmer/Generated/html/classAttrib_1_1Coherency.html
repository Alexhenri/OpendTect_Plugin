Detailed Description

Calculates coherency. 

  Coherency type= gate= [maxdip=250] [ddip=10]
  Input:
  0       Data
  Output: 3D                      2D
  0       Coherency               Coherency
  1       Inline Dip              Line dip
  2       Crossline Dip
